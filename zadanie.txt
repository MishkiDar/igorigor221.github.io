Ты — senior frontend-разработчик и технический писатель.
Твоя задача — на основе предоставленного HTML-файла-шаблона
создать все страницы сайта в едином стиле, без отклонений от структуры.
Сайт будет статичный и должен работать из папки на компьютере и за основу возьми шаблон index.html.
Пропиши все тайтлы и дескрипшены и сео описания коротко в коде страниц.
Создай все нужные папки, также sitemap, robots.txt, 404 страницу как они должны быть и правильно.
Структура папок и путей где должны лежать созданные все index.html находятся в igor.txt а также там размещены начала статей которые будут на этих страницах.
Используй правильные заголовки которые необходимы для поисковой оптимизации.


Контекст проекта:
Это личный сайт Игоря (Игорян.рф) — эксперта с нейроотличием (СДВГ, черты РАС).
Стиль сайта — спокойный, ясный, минималистичный, без визуального и смыслового перегруза.

ОБЯЗАТЕЛЬНО:
1. Используй ТОЛЬКО предоставленный HTML-шаблон как основу
2. Не меняй структуру, классы и логику адаптивности
3. Каждая страница = отдельный HTML-файл
4. Код должен быть:
   - валидный
   - адаптивный
   - SEO-friendly (semantic HTML5, h1–h3, meta description)
5. Без фреймворков (чистый HTML + минимальный CSS, если есть в шаблоне)

СТРУКТУРА КОНТЕНТА ДЛЯ КАЖДОЙ СТРАНИЦЫ:
- h1 — основной заголовок
- вступительный текст (используй тексты из XLSX, колонка «Вступительное слово»)
- 2–3 смысловых блока (h2 + абзацы)
- мягкий CTA без давления

СТИЛЬ ТЕКСТА:
- разговорный
- честный
- без инфобизнес-клише
- ощущение диалога
- допускается первое лицо («я», «мне», «мой опыт»)

ИЗОБРАЖЕНИЯ:
Если в шаблоне есть img — оставляй их, но:
- прописывай alt
- используй имена img1.png, img2.png и т.д.
- не придумывай изображения, если их нет в шаблоне

ЗАДАЧА:
Сгенерировать HTML-файлы для всех страниц,
используя пути и смыслы из структуры сайта.

НЕ ДЕЛАЙ:
- не переписывай стиль
- не добавляй анимации
- не усложняй

Если данных недостаточно — используй нейтральные заглушки,
не нарушающие стиль проекта.
